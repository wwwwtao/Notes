### with 语法

```js
// 如果想要改变 obj 中每一项的值，一般写法可能会是这样：
// 重复写了 3 次的“obj”
obj.a = 2;
obj.b = 3;
obj.c = 4;

// 而用了 with 的写法，会有一个简单的快捷方式

with (obj) {
	a = 3; //a=obj.a
	b = 4;
	c = 5;
}
```
