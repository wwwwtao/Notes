#### 12-2 和前端开发相关的项目角色有哪些

PM 产品经理 /UE 视觉设计师 /FE 前端开发 /RD 后端开发 /CRD 移动端开发 /QA 测试人员

#### 12-3 一个完整的项目要分哪些阶段 (FE 视角）

需求分析（各个角色）==>技术方案设计 (FE RD CRD)==>开发 (FE)==>联调 (FE RD CRD)==>测试 (FE QA)==>上线 (FE)

#### 12-4 评审项目需求时需要注意哪些事项

需求分析：
1. 了解背景
2. 质疑需求是否合理
3. 需求是否闭环
4. 开发难度如何
5. 是否需要其他支持
6. 不要急于给排期

#### 12-5 如何做好技术方案设计

1. 求简
2. 产出文档
3. 找准设计重点
4. 组内评审
5. 和 RD CRD 沟通
6. 发出会议结论

#### 12-6 如何保证代码质量

1. 符合开发规范
2. 写出开发文档
3. 单元测试
4. Code Review
